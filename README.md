# Feature-Selection-using-RFE-and-Classification-with-KNN-on-Iris-Dataset

This project demonstrates how to perform feature selection and classification using machine learning techniques in Python. The Iris dataset is used to identify the most important features and build a predictive model.
The workflow includes:
Data preprocessing using Pandas and NumPy
Feature selection using Recursive Feature Elimination (RFE)
Splitting the dataset into training and testing sets
Feature scaling using StandardScaler
Target encoding using LabelEncoder
Model training using K-Nearest Neighbors (KNN) classifier
Making predictions on the test dataset
The goal of this project is to understand how feature selection improves model performance and reduces dimensionality in machine learning workflows.
